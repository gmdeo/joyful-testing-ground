# Hermes Agent + Lovable Integration Workflow

## The Strategy

This repo is connected to Lovable with bidirectional sync:
- Edits in Lovable → GitHub  
- Edits in GitHub (via Hermes Agent) → Lovable

**Zero Lovable build credits consumed** when using Hermes Agent for coding!

## How it Works

1. **Hermes Agent edits code directly in this GitHub repo**
   - Uses standard git operations
   - No Lovable AI interactions
   - No build credits consumed

2. **Lovable syncs changes automatically**
   - GitHub integration watches the main branch
   - Changes sync to Lovable platform
   - All Lovable features still work

3. **Lovable connections remain active**
   - Supabase integration (shared connector)
   - All other Lovelove integrations
   - Hosting infrastructure
